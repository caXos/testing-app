1. Clone project 
2. Rename the file .env.example as .env
3. Config Database settings
Example: 
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=DEV_nwta
DB_USERNAME=newswiretestinguser
DB_PASSWORD=newswiretestingpswd

4. Open in IDE (e.g.: VisualStudioCode)
5. Open a terminal
6. Run composer install To install dependencies
7. Run npm install To install npm packages
8. Run php artisan breeze:install vue to install Laravel Breeze Starter Kit, with Vue, Vite and Inertia files
9. Run php artisan migrate To initialize the pgsql database
10. Run npm install again to update with breeze, vite, vue and inertia files
11. Run npm run dev or npm run build to build and to watch for changes replace build with dev
12. Open a browser and go to http://localhost:8080 and register an account, on the dashboard page you can work on the task
